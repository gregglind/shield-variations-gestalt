Help Us Make The Future of Firefox

Benefits

Improved Firefox performance. Try good features first.

Risks

Very small risk of decreased performance. Uninstalling addon will fix it.

Data Collected

This addon records if the user was online during the session, along with basic environment data. Records end of study, or if user uninstalls. Opens a survey at experiment end in 7 days.

About the study

  • Addon Author: Gregg Lind (glind@mozilla.com)
  • Data Lead: Gregg Lind (glind@mozilla.com)
  • Goal: determing which performance changes affect user perception.
  • Duration: 7 days.
